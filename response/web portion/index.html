This project was focused on Markup, and the history of old fashion printing. In the begining of this project I made a physical copy of my content by cutting out found type from magazines to create a small response to the video "Farewell - Etaoin Shrdlu 1978" which is a film about the last working days of hot metal press typesetting. It is directed by Carl Schlesinger and David Loeb Weiss, and they recorded the last time this metal type set was used in the New York Times printing facility.
